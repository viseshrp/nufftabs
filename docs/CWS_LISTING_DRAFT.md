# Chrome Web Store Listing Draft

## Name
nufftabs

## Summary (132 chars max)
Condense open tabs into local saved groups and restore them anytime. No tracking, no analytics, and no remote data collection.

## Description
nufftabs helps you clear tab overload without losing context.

- Condense all open tabs with one click
- Restore one tab, a whole group, or all groups
- Keep everything local using `chrome.storage.local`
- Import/export JSON backups
- Import from OneTab exports
- Optional memory-saving restore mode
- Theme controls: System, Light, Dark

Privacy-first by design:
- No analytics
- No tracking
- No external API calls
- No host permissions

## Category
Productivity

## Language
English (United States)

## Permissions Justification
- `tabs`: Required to read open tab URLs/titles, close condensed tabs, and restore tabs/windows.
- `storage`: Required to store saved tab groups and extension settings locally on-device.
